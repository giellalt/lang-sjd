LEXICON adv
+Adv: # ;

LEXICON Adverb
! lexical adverbs
чофта adv "много / очень / much" ;
четк adv "насмерть / deadly" ;

! ?derived adverbs from adjectives; suffix /-as/; weak grade, depalatalization
маӈас adv "назад / back (direction)" ; !derived from маӈӈь 'задний'
ва̄нас adv "мало / some" ; !derived from ва̄ннҍ 'мало / some (adj)'

! ?derived adverbs from nouns; suffix /-el't/; weak grade
нюэзельт adv "плохо / badly" ; !derived from нюэссь '(что-то) плохое'
ла̄шькэльт adv "лениво / laizy" ; !derived from ла̄шьк 'лень'

! ?derived adverbs from adjectives; suffix /-tenn'e/; strong grade
! historically, the /-t-/ must have been a coaffix (probably deriving a verb), that's why the strong grade appears 
шӣгктэнне adv "хорошо / well" ; !derived from шӣг 'хороший'
вэххтэнне adv "быстро / fast" ; !derived from вэхтэсь 'быстрый:attr'

! ?derived adverbs from (derived) verbs; suffix /-enn'e/; no gradation
ӣннъенне adv "громко / loud" ; !derived from ӣннъе 'звучать' (< ӣнн'голос')
роаммъенне adv "радостно / gladly" ; !derived from ?роаммъе (< роамм 'радост')
